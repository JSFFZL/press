# 组件库

  • Vant： 轻量、可靠的移动端 Vue 组件库 （https://youzan.github.io/vant/）

  • Zan Proxy：本地代码调试线上页面，环境再也不是问题 （https://youzan.github.io/zan-proxy/）

  • Zent ( \ˈzent\ ) 是有赞 PC 端 WebUI 规范的 React 实现，提供了一整套基础的 UI 组件（https://youzan.github.io/zent/zh/guides/install）

  • Felint： 好用的前端静态代码校验工具包

  • Astroboy：基于koa2的Node.js SFB框架

  • Show Me The Code：多人在线 Coding 平台
